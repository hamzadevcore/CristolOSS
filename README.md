# ◈ CRISTOL TERMINAL ◈

Cristol started as a single AI prompt.

> “Roleplay as a writer, and I am the main character. I control the character. You control the story.”

That was 6 months ago.

Now it's a full application. Backend included. Rolling summaries. Chunk saving. System prompts. Stories that remember things they probably shouldn't.

Cristol Terminal is built for longform AI storytelling and persistent roleplay without the context window collapsing into dust halfway through chapter three.

---

## Features

* Infinite-ish context through rolling summaries
* Chunk-based memory saving
* Persistent story sessions
* Custom system prompts
* Terminal-style interface
* Backend support
* Probably emotionally attached to your characters already

---

## Setup

Download and run the ```installer.exe```, and it'll set up the rest! It's that simple!

---

## What It's Good At

* Longform roleplay
* Interactive fiction
* Character persistence
* Slow-burn conversations
* Making you say “wait how did it remember that”

---

## Philosophy

Most AI chats forget everything eventually. Entropy, heat-death and all that.

Cristol doesn't like doing that.
